# LearnPath AI - AI-Powered Learning Path Generator

A full-stack web application that generates personalized learning paths using an AI-powered recommendation engine. This project demonstrates skills in frontend development, backend API design, and AI integration.

## Features

- **Personalized Learning Paths**: AI generates custom learning paths based on user goals, current level, and time availability
- **Interactive Visualization**: Clean, responsive UI showing the complete learning journey
- **Smart Recommendations**: Resources are curated based on user interests and learning style
- **Progress Tracking**: Milestones and weekly goals to keep learners on track
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **AI Engine**: Simulated AI with ready integration points for OpenAI/Google AI APIs
- **Architecture**: RESTful API design, MVC pattern
- **Deployment Ready**: Configured for Heroku, AWS, or similar platforms
