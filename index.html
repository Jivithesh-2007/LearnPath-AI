Project Overview

LearnPath AI is a full-stack web application that generates personalized learning paths using an AI-powered recommendation engine. It demonstrates skills in frontend development, backend API design, and AI integration.

Key Features

  • AI-powered personalized learning path generation
  • Interactive visualization of learning journey
  • Dynamic resource recommendations based on user profile
  • Progress tracking and milestone planning
  • Responsive design for all devices
